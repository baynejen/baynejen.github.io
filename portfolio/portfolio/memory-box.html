Memory Box

UX, UI, HTML, CSS, Javascript

A personal project to archive the hundreds of ticket stubs I've accumulated (some might say hoarded) over the years. A great way to remember the amazing gigs, exhibitions and places I've been lucky to go to. Built as part of the 2015 General Assembly Web Design Circuits.

The brief

Create a full responsive website that uses HTML and CSS. Follow the standards of web design - hierarchy, whitespace, flow - and create an intuitive navigation.

Defining the user experience

I wanted to keep the site architecture simple, with a menu that allows the user to drill down by content type. Most work was dedicated to the wireframing stage before proceeding to development. This process involved rounds of simplication before I was happy that that structure was as simple as possible. I used balloon logo as a navigational element to take the user to the index page.
