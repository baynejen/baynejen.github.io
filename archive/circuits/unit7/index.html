My progress

  • Greater appreciation of design principles, especially the importance of hierarchy, flow, scale and space. A new-found love of letter spacing and line height.
  • Advanced page layout, using columns and floats...especially when they work!
  • Learning to structure a webpage with appropriate semantic tags, including header and footer.
  • Exprimenting with multiple ways to style a nav bar and use it as the foundation for a coherent page layout.
  • Adding interactive elements using pseudo-classes and learning to refine a design through small details.
  • Being able to design through code rather in photoshop!

My struggles

  • Learning to keep it simple and not overcomplicate a design.
  • Taking the time to work through ideas on paper first, to properly research and test ideas before building final wireframes.
  • Building confidence in choosing matching typefaces and colour schemes...a struggle I've enjoyed.
  • Building fully responsive designs that adapt to any screen, and dealing with the unpredictability of floats and columns.
  • Aligning elements vertically so that they sit comfortably within a parent element.
  • Keeping track of divs within divs (within divs) and making sure to order the classes and IDs in the correct order to preserve parent/child relationships.

Design critique: Square Space

This site showcases large images with bold typography, using ample whitespace to draw attention to the visuals and headlines. A minimal colour palette compliments this clean layout, whilst the smooth interactive scrolling makes the experience playful.

Typography: Good contrast is created between the titles and body copy with large and small font sizes and dark and lighter font colours. Sub-titles are then called out with smaller, but uppercase font. This contrast helps create hierarchy, which aids flow.

Navigation + Flow: As someone who had never visited the site before I found the overall navigation a little difficult. I was unsure whether the main action should be scrolling down, cliking on the main nav, or on the side nav. Each link on the top nav houses its own unique content, and I think that the site layout would become more intuitive if the top nav either linked directly to content already present on the main page, or if there wasn't such a long vertical scroll on the homepage.

Furthermore, when scrolling downwards I felt there were too many types of scrolls and intractivity on this page; on one section the images and text flip within one computer screen; on another they appear asymmetrically as you scroll downwards. I think these different effects can distract from the overall site objective to convert new users into subscribers.

Having said that, I think that because the design is so clean - the designer can have a little more flair with the navigation. The large images and titles and ample whitespace help punctuate and guide the eye downwards, creating a sense of flow.

I like that the main cta jumps to the nav as you scroll, as it reminds the users what the one core action of that site is: to get started. It makes it clear that this site is a service, not a portfolio site.

Design critique: Space Jam

I love the way the navigation mirrors the centre graphic. It draws the user's eye in and makes it an interactive experience. I would love to see some pseudo-class applied to each nav link when hovered over to amplify this playful experience.

I also love how the background image creates a skeumorphic effect - I really do believe I'm in outer space!

Typography: the nav's readability would be improved with larger font size and increased letter spacing. Furthermore, the links are too close to the images - increasing margins here would really aid overall comprehension. Furthermore, I think the red footer text doesn't compliment with the yellow link colour.

Overall there is little to no structure on this page - which is perhaps the point, as every element feels like it's floating around in space. Anchoring the page with a top nav bar and final footer bar would make it easier to read through the nav links from left to right, rather than round in circles. Some additional content on the homepage would also aid the flow and provide some clarity as to what the site's main purpose is.

The layout and styling of each page is completely different, there is a real lack of consistency which makes it hard to understand where in the site you are. At one point you even seem to be on a completely different site! Having a consistent header and nav across all pages would improve the site nagivation.
